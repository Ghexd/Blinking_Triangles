# Blinking Triangles

This is a live wallpaper that reacts to system audio.

It's designed to work with [Lively Wallpaper](https://github.com/rocksdanister/lively).

I took inspiration from [this](https://codepen.io/msurguy/pen/wvGgzN) triangular pattern (so check it out, it's really cool).
